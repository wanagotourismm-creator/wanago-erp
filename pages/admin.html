📬 End of Day Report (EODR) — Auto Email + WhatsApp
Sends a full report to Admin & Branch Managers, and a personal report (their own data only) to each team member — every day at the chosen time, via Gmail + WhatsApp.

After clicking "Setup Daily Trigger", the Apps Script will automatically send reports every day at the selected time — even when your browser is closed.

Quick Setup Guide
Step-by-step for each integration
📊 Meta Ads
  1. Open Meta Business Suite
  2. Go to System Users → Create Admin user
  3. Generate token with ads_read
  4. Copy Ad Account ID (act_…)
  5. Paste above → Save → Test
💬 WhatsApp API
  1. Go to Meta for Developers
  2. Create a WhatsApp Business App
  3. Copy Phone Number ID
  4. Generate System User token
  5. Add Business Account ID → Test
📧 Gmail OAuth2
  1. Go to Google Cloud Console
  2. Enable Gmail API
  3. Create OAuth 2.0 Client ID
  4. Add your domain as authorized origin
  5. Paste Client ID → Save → Authorize
📊 Google Sheets
  1. Create a new Google Sheet
  2. Extensions → Apps Script
  3. Paste the Code.gs provided below
  4. Deploy → New deployment → Web App
  5. Execute as: Me · Access: Anyone
  6. Copy URL → Paste above → Save → Test
🔔 Push Notifications
  1. Firebase Console → Project Settings
  2. Cloud Messaging → Web configuration
  3. Generate key pair → Copy public key
  4. Paste VAPID key above → Save
  5. Each team member clicks 🔕 in notification bell
  6. Tokens auto-register in Firestore
